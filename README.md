
# Temporizador

Um temporizador simples, onde você pode adicionar os minutos e ao finalizar a contagem ele emite um alerta sonoro. 
No canto inferior direito existe um botão que permite executar um efeito sonoro.



## Stacks utilizadas

1. HTML
2. CSS
3. JavaScript
